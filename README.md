# Recursive Turtle Art

A Python Turtle Graphics project that generates recursive geometric patterns using circles, lines, and rotations.

## Overview

This project uses Python's `turtle` module and recursion to create complex artistic patterns.

The drawing is generated by repeatedly:

- Moving the turtle forward
- Drawing circles
- Rotating by a fixed angle
- Making recursive calls
- Combining multiple branches to create fractal-like structures

The result is an abstract recursive design that demonstrates the power of recursion in computer graphics.
